# NovelSharing
It is Web Application to exchange Novels.


## Tech Stack
-  Java
-	Servlet
-	JSP
-	CSS
-	JavaScript
-	MYSQL Database
-	Glassfish Server


## IDE
Netbeans IDE 8.2
